what's this?

Centauri is a decentralized p2p network which allows users around the world to send 100% encrypted and private files, messages, and more, without relying on conventional internet infrastructure. You are connected to a Centauri network peer. This does not have internet access.
You can use a Centauri client on your device, or one of the services below to access the Centauri network.

What does this do?
Centauri Edge peers are intended to service areas with little to no internet access. A local Centauri network can be set up with edge peers forming a mesh network through which local clients can communicate with each other over long distances.
